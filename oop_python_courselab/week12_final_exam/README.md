# Inventory Management System (Week 12 Final Exam)

Description
- A simple object-oriented inventory system for a small store.
- Uses classes, encapsulation, file storage, and exception handling.

Features
- Add Product
- View Products
- Search Product by ID
- Update Product Quantity

How to run
1. Open a terminal in the `week12_final_exam` folder.
2. Run:

```
python main.py
```
