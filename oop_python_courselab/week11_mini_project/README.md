# Student Information System (Week 11 Mini Project)

Description
- A simple menu-driven Student Information System demonstrating OOP, file handling, and exception handling.

Features
- Add Student
- View Students
- Search Student by ID

How to run
1. Open a terminal in this folder.
2. Run:

```
python main.py
```
