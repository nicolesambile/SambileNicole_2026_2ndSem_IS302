# Object-Oriented Programming using Python Course

This repository contains exercises and projects for the BS Information Systems Object-Oriented Programming course using Python.

## Structure

- `week01_intro/`: Introduction to Python basics
- `week02_control_structures/`: Control structures and loops
- `week03_functions/`: Functions and modular programming
- `week04_data_collections/`: Lists, dictionaries, and data collections
- `week05_oop_basics/`: Introduction to object-oriented programming

## Setup

Ensure you have Python 3.x, VS Code, Git, and a GitHub account installed.
